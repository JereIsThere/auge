1Password CLI — die wichtigsten Commands

op signin                              # Session starten
op vault list                          # Vaults anzeigen
op item list --vault Private           # Items eines Vaults
op item get "GitHub" --fields password # Einzelnes Feld
op item get "GitHub" --otp             # Aktuellen TOTP-Code
op item create --category=login \
   --title="Acme" \
   --vault=Work \
   username=max@example.com \
   password=...                        # Item anlegen
op read "op://Work/Stripe/api_key"     # Secret-Reference auflösen
op run --env-file=.env -- npm start    # mit env vars starten
op inject -i tpl.yaml -o out.yaml      # Template rendern
op document get "Pass" --output=p.pdf  # Datei-Anhang abrufen
op plugin init gh                      # Plugin für CLI-Tool
op signout                             # Session beenden

Secret-Reference-Syntax

op://<Vault>/<Item-Title-or-UUID>/<Field>
op://<Vault>/<Item>/<Section>/<Field>     # mit Section
op://Work/Stripe/api_key
op://Private/GitHub/credential
op://Work/Prod DB/Connection/host
op://Private/SSH GitHub/private_key

Item-Typ → wann?

LoginAccount mit Username/Password — Default
Passwordnur Passwort, kein Account
Secure NoteFreitext / Markdown
Credit CardKarten
IdentityAdresse für Form-Autofill
API CredentialAPI-Keys, Tokens
SSH KeySSH-Schlüssel — mit SSH-Agent
Database / Serverstrukturierte Connections
Software LicenseLizenzkeys + Bestelldatum
DocumentPDFs, Scans

Watchtower-Kategorien

Compromisedim Leak aufgetaucht → sofort rotieren
VulnerableService hatte Breach → rotieren
Reusedidentisch auf mehreren Sites
Weakgeringe Entropie
Inactive 2FA2FA nicht aktiviert
ExpiringAblaufdatum nähert sich

SSH-Agent-Setup (Recap)

# ~/.ssh/config
Host *
  IdentityAgent ~/Library/Group\ Containers/2BUA8C4S2C.com.1password/t/agent.sock
# Linux:   ~/.1password/agent.sock
# Windows: \\.\pipe\openssh-ssh-agent

Git-Commit-Signing (Recap)

git config --global gpg.format ssh
git config --global user.signingkey "ssh-ed25519 AAAA..."
git config --global commit.gpgsign true
git config --global gpg.ssh.program \
  "/Applications/1Password.app/Contents/MacOS/op-ssh-sign"

Hygiene-Routine

Alle 3 Monate: Watchtower durchgehen, Compromised + Reused leeren, tote Accounts löschen.

Alle 6 Monate: Audit (Geräte-Liste, Recovery-Kontakt, Emergency Kit, YubiKey-Test).

Bei jedem Job-Wechsel: Privates aus dem Work-Account exportieren, Service-Account-Tokens rotieren.
