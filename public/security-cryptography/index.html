💡 Buffer Overflow (konzeptionell)

// Verwundbar: kein Bounds-Check

strcpy(buffer, user_input);  // user_input > 8 Bytes → Stack Overflow!

// Sicher:
strncpy(buffer, user_input, sizeof(buffer) - 1);
