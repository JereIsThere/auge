💡 Einfache Stack Machine

class VM:
    def __init__(self):
        self.stack = []
    
    def execute(self, instructions):
        for op, *args in instructions:
            if op == 'PUSH':   self.stack.append(args[0])
            elif op == 'ADD':  self.stack.append(self.stack.pop() + self.stack.pop())
            elif op == 'PRINT':print(self.stack.pop())

vm = VM()
vm.execute([('PUSH', 3), ('PUSH', 4), ('ADD',), ('PRINT',)])  # → 7

💡 Warum JIT so schnell ist

Interpreter:  Source → [parse] → AST → [evaluate] = langsam
AOT Compiler: Source → [compile] → Binary → [run] = schnell, aber unflexibel
JIT:          Source → Bytecode → [run] → HotSpot detected → [compile to native] → schnell UND flexibel
