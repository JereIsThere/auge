💡 Your First Algorithms

Array Traversal

def print_array(arr):
    for element in arr:
        print(element)
# Time: O(n), Space: O(1)

Linear Search

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
# Time: O(n), Space: O(1)

Simple Bubble Sort (Beginner)

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
# Time: O(n²), Space: O(1)
