## 💡 Common Algorithms

- **Merge Sort**: O(n log n) stable sorting
- **Quick Sort**: O(n log n) average, in-place
- **Binary Search**: O(log n) on sorted arrays
- **DFS**: O(V + E) graph traversal
- **BFS**: O(V + E) shortest path in unweighted graphs
