# LaTeX for Intermediate 🟡

Level up with custom commands, bibliography management, and professional document structure.

## Key Concepts

| Concept | Purpose | Example |
|---------|---------|---------|
| `\newcommand` | Create custom commands | `\newcommand{\mycmd}{text}` |
| `\usepackage` | Import packages | `\usepackage{graphicx}` |
| BibTeX | Manage citations | `\cite{key}`, `\bibliography{refs}` |
| `\includegraphics` | Insert images | `\includegraphics[width=5cm]{fig.png}` |
| `\caption` | Figure captions | `\caption{Figure description}` |
| `\label` & `\ref` | Cross-references | `\label{fig:1}` then `\ref{fig:1}` |
