# LaTeX 📝

Professional document preparation and typesetting. Create beautiful research papers, theses, and technical documents.

**Note**: All resources here are **online-focused** — use Overleaf or TeXLive Online, no local installation required!

## Why Learn LaTeX?

- **Professional Quality**: Used in academia and publishing worldwide
- **Math & Science**: Unmatched for mathematical equations and scientific notation
- **Consistency**: Separates content from formatting
- **Reproducibility**: Documents are version-controlable and portable
- **Cs Skill**: Your CS classmates will expect you to know this!

## Learning Path

### Level Progression
```
Beginner      → Document structure, basic commands, Overleaf basics
              → Write a simple article or resume

Intermediate  → Custom commands, packages, bibliography management
              → Multi-page documents, figures, tables

Advanced      → Custom packages, TikZ graphics, templates
              → Complex scientific documents, automating publishing
```

## Quick Tips

1. **Use Overleaf**: [overleaf.com](https://www.overleaf.com) is the easiest way to start
2. **Start with Templates**: Don't build from scratch — use Overleaf's templates
3. **Learn by Doing**: Type along with tutorials
4. **Read Examples**: Study other people's .tex files
5. **Google is Your Friend**: LaTeX has massive community support
