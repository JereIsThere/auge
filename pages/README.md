# Claude Learnings 📚

Curated learning resources for advanced programming concepts with high skill ceilings. Each topic is organized by difficulty level (Beginner → Intermediate → Advanced) with interactive resources, practical examples, and hands-on exercises.

## Topics

### 1. **Regular Expressions (RegEx)** 🔍
Master pattern matching and text processing. From basic metacharacters to advanced lookahead/lookbehind techniques.
- **[RegEx Learning Path →](./regex/README.md)**

### 2. **LaTeX** 📝  
Professional document preparation and typesetting. All resources are **online-focused** (Overleaf, TeXLive Online) — no local installation required!
- **[LaTeX Learning Path →](./latex/README.md)**

### 3. **Data Structures & Algorithms** 🏗️
Fundamentals of efficient programming. Arrays, Trees, Graphs, Sorting, and complexity analysis.
- **[Data Structures & Algorithms Learning Path →](./data-structures-algorithms/README.md)**

### 4. **TensorFlow & Keras (Python)** 🤖
Machine Learning fundamentals and neural networks. Perfect preparation for ML courses at university.
- **[TensorFlow/Keras Learning Path →](./tensorflow-keras/README.md)**

## How to Use This Repository

1. Pick a topic from the list above
2. Start with **Beginner** level if you're new to the topic
3. Follow the curated resources in order (prioritized by quality)
4. Work through the **examples/** folder for hands-on practice
5. Progress to Intermediate and Advanced as you master each level

## Tips for Learning

✨ **Interactive First** - Use interactive platforms (regex101.com, Overleaf, VisuAlgo.net) to experiment in real-time

🎯 **Projects Over Theory** - Build small projects at each level to solidify understanding

💾 **Track Progress** - Each topic includes checkpoints and practice exercises

🔗 **Follow Links** - All resources are carefully selected for quality and relevance
