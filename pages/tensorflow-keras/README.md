# TensorFlow & Keras (Python) 🤖

Machine Learning fundamentals and neural networks. Perfect preparation for ML courses!

## Why Learn This?

- **AI/ML Foundation**: Essential for modern ML applications
- **University Prep**: Prepare for university ML courses (you mentioned!)
- **Practical Skill**: Build real neural networks and train models
- **Industry Relevant**: TensorFlow/Keras is industry standard
- **High Skill Ceiling**: From basic networks to cutting-edge research

## Learning Path

### Level Progression
```
Beginner      → Python basics, NumPy, Pandas fundamentals
              → TensorFlow setup, first neural network, MNIST

Intermediate  → Convolutional Neural Networks (CNN)
              → Recurrent Neural Networks (RNN/LSTM)
              → Model evaluation, training optimization

Advanced      → Custom layers and models
              → Transfer learning, fine-tuning
              → Deployment, TensorFlow optimization
```

## Quick Tips

1. **Python First**: Get comfortable with Python before TensorFlow
2. **Jupyter Notebooks**: Use Colab (free GPU) for experimentation
3. **Dataset Size Matters**: Start with small datasets (MNIST, CIFAR-10)
4. **Understand Theory**: Don't skip the math behind neural networks
5. **Iterate Quickly**: Experiment with architectures and hyperparameters

## Prerequisites

- Python 3.7+ (comfortable with syntax)
- Basic math (linear algebra, calculus)
- NumPy and Pandas basics

## Common Use Cases

| Use Case | Difficulty | Topic |
|----------|-----------|-------|

| CIFAR-10 image classification | Intermediate | Convolutional networks |
