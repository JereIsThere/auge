# TensorFlow & Keras for Intermediate 🟡

Master Convolutional and Recurrent neural networks for image and sequence data.

## Key Concepts

| Concept | What It Does | Best For |
|---------|-------------|----------|
| Convolutional Layer | Extracts local features | Image data |
| Pooling | Reduces spatial dimensions | Image compression |
| LSTM/GRU | Remembers sequential context | Time series, text |
| Dropout | Prevents overfitting | Regularization |
| Batch Normalization | Stabilizes training | Deeper networks |

## 💡 Common Architectures

- **AlexNet / VGG**: Image classification foundations
- **ResNet**: Residual networks, skip connections
- **Inception**: Multi-scale feature extraction
- **LSTM Networks**: Sequential data processing
- **Sequence-to-Sequence**: Encoder-decoder models

## Datasets to Practice With

- **CIFAR-10**: 60K tiny images (10 classes)
- **ImageNet**: Large-scale image classification
- **MNIST Fashion**: Fashion item classification
- **IMDB Reviews**: Text classification
- **Time Series**: Stock prices, weather data
